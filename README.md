# 阿坝州云硕信息技术有限公司报账资料制作系统

## 项目简介

本系统是一个基于Web的报账资料制作工具，主要用于生成购销合同、询价单、送货单等报账所需的各类文档。系统采用纯HTML、CSS和JavaScript实现，无需安装任何依赖，可直接在浏览器中运行。

## 功能特性

### 核心功能

1. **合同生成**
   - 支持填写甲方、乙方信息
   - 自动计算货物总金额
   - 生成标准格式的购销合同
   - 导出为Word文档

2. **询价管理**
   - 生成询价单
   - 自动计算三家公司的报价
   - 生成报价清单
   - 支持复制报价信息

3. **送货单生成**
   - 自动采集货物信息
   - 生成标准格式的送货单
   - 导出为Word文档

4. **数据管理**
   - 导入Excel/Word文件中的货物信息
   - 清空所有数据
   - 资料成套功能

### 技术特点

- **纯前端实现**：无需后端服务器，所有操作在浏览器中完成
- **实时计算**：货物金额和总金额实时计算
- **数据持久化**：自动保存甲乙双方信息
- **现代化界面**：采用玻璃态设计，美观易用
- **多格式导出**：支持导出为Word文档

## 快速开始

### 运行方式

1. 下载完整项目文件
2. 直接在浏览器中打开 `合同生成器.html` 文件
3. 开始使用系统功能

### 系统要求

- 现代浏览器（Chrome、Firefox、Edge等）
- 互联网连接（首次加载需要下载第三方库）

## 使用指南

### 1. 填写合同信息

- **项目名称**：填写项目的名称
- **询价日期**：选择询价的日期
- **甲方信息**：填写采购方的公司名称、联系人、联系电话
- **乙方信息**：填写销售方的公司名称、联系人、联系电话

### 2. 管理货物清单

- **添加货物**：点击货物行中的 `+` 按钮
- **删除货物**：点击货物行中的 `-` 按钮
- **填写货物信息**：
  - 名称：货物的名称
  - 单位：货物的计量单位（支持下拉选择）
  - 数量：货物的数量
  - 单价：货物的单价
  - 金额：自动计算，无需手动填写
  - 备注：货物的备注信息

### 3. 生成文档

- **下载合同**：生成并下载购销合同
- **询价生成**：生成询价单
- **询价计算**：计算三家公司的报价并显示结果
- **生成送货单**：生成送货单
- **资料成套**：打开资料成套页面
- **清空数据**：清空所有填写的数据

### 4. 导入货物信息

- 点击 **导入商品** 按钮
- 选择 Excel 或 Word 文件
- 系统会自动提取文件中的货物信息并填充到货物清单中

## 功能模块详解

### 合同生成模块

生成标准格式的购销合同，包含：
- 甲乙双方信息
- 合同条款
- 货物清单（名称、单位、数量、单价、金额）
- 总金额（小写和大写）
- 签字盖章区域

### 询价管理模块

- **询价生成**：生成包含货物清单的询价单
- **询价计算**：
  - 自动为三家公司生成报价
  - 第二、三家公司报价比原始价格高5%-15%
  - 第一次报价比最终报价高5%-20%
  - 生成报价清单，包含第一次和最终报价
  - 显示报价结果弹窗，支持复制信息和下载报价单

### 送货单模块

生成标准格式的送货单，包含：
- 项目名称和送货日期
- 供应商和收货单位信息
- 货物清单
- 合计金额
- 签字盖章区域

### 资料成套模块

打开 `kkxx/index.html` 页面，提供更多报账相关的资料模板和工具。

## 技术实现

### 核心技术

- **HTML5**：页面结构
- **CSS3**：样式设计（玻璃态效果）
- **JavaScript**：业务逻辑
- **SheetJS**：Excel文件解析
- **Mammoth**：Word文件解析

### 目录结构

```
合同生成器/
├── 合同生成器.html    # 主页面
├── style.css         # 样式文件
├── favicon.webp      # 网页图标
└── kkxx/             # 资料成套文件夹
    ├── index.html    # 资料成套主页面
    ├── Word/         # 文档模板
    └── data/         # 数据文件
```

### 关键功能实现

1. **实时计算**：通过 `calculateTotal()` 函数实时计算货物金额和总金额
2. **数据导入**：使用 SheetJS 和 Mammoth 库解析 Excel 和 Word 文件
3. **文档导出**：通过 Blob 对象生成 Word 文档并下载
4. **数据持久化**：使用 localStorage 保存甲乙双方信息
5. **报价计算**：使用随机数生成不同公司的报价

## 常见问题

### 1. 导入文件失败

- 确保文件格式正确（Excel：.xls, .xlsx；Word：.doc, .docx）
- 确保文件中包含正确的货物信息格式
- 检查浏览器控制台是否有错误信息

### 2. 清空数据后无法添加货物

- 系统会自动保留一个默认的货物行，可直接在其中填写信息
- 点击 `+` 按钮添加新的货物行

### 3. 报价计算结果不一致

- 报价计算使用随机数生成，每次计算结果会有所不同
- 系统会确保第二、三家公司报价高于原始价格
- 第一次报价会高于最终报价

### 4. 导出的Word文档格式问题

- 导出的文档使用了标准的Word格式
- 确保使用Microsoft Word或兼容的文档查看器打开
- 如遇格式问题，可尝试调整浏览器设置

## 注意事项

1. **数据安全**：本系统在本地运行，所有数据存储在浏览器中，不会上传到任何服务器
2. **浏览器兼容性**：建议使用Chrome、Firefox、Edge等现代浏览器
3. **文件大小**：导入的Excel或Word文件不宜过大，建议不超过10MB
4. **数据备份**：重要数据请及时导出备份，浏览器清除缓存可能会导致数据丢失
